# Product Images

Add your product images here. The following files are expected:

- `headphones.jpg` - Wireless Headphones product image
- `watch.jpg` - Smart Watch product image
- `shoes.jpg` - Sports Shoes product image
- `case.jpg` - Phone Case product image
- `backpack.jpg` - Laptop Backpack product image
- `cable.jpg` - USB-C Cable product image
- `speaker.jpg` - Portable Speaker product image
- `tshirt.jpg` - T-Shirt Pack product image

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Size**: Recommended 400x400px or larger
- **Aspect Ratio**: Square (1:1)
- **File Size**: Keep under 500KB for faster loading

## How to Add Images

1. Download or create product images
2. Save them with the filenames mentioned above
3. Place them in this folder (`public/images/products/`)
4. The app will automatically use them without any code changes
